Practice 3

Create the following Test:

* Automate 5 Smoke test

1. Verify all Project are getting

2. Verify all items are getting

3. Verify the user information is getting

4. Verify a item can be retrievec

5. Verify a Project can be created successfully

* Automate 5 Sanity test

1. Verify information (email& fullname) about user is getting correctly

2. Verify Priority field on specific item is returned correctly

3. Verify Projects information are listed properly

4. Verify after a filter is applied then, the items are listed properly

5. Verify information about icons is displayed properly

* Automate 10 workflows(include negative tests)

1. Verify a new Project can be created

2. Verify a new Item can be createad as expected

3. Verify information to specific Project is displayed as expected

4. Verify information to specific item is displayed as expected

5. Verify name to specific project can be updated

6. Verify Pre-fij to all name items on a proyect can be added

7. Verify a project can be removed

8. Verify an item can be deleted

9. Verify an error is displayed if email is updated with invalid email

10. Verify an error is displayed trying to create an item with invalid id of proyect

Notes:

For workflows, please consider:

• Automating also the pre-requisite steps

• Do not leave buggy data in the app. For instance, If the test needs 5 projects created, create them and once test is done, delete them

• Apply jasmine-node matchers
